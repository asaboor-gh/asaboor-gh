### About

I use density functional theory (DFT) to predict the electronic properties of dilute bismide III–V alloys and strained transition-metal dichalcogenide monolayers — targeting mid-infrared, valleytronic, and topological applications. I also build open-source Python tools for interactive DFT workflows in Jupyter.

**Advisor:** [Prof. Anderson Janotti](https://mseg.udel.edu/people/janotti/) · Condensed Matter Theory Group

### Research in Progress

| Project | Method | Key Idea |
|---------|--------|----------|
| Dilute bismide quaternary alloys | HSE06 + SOC | InGaAsBi / InAlAsBi lattice-matched to InP for telecom & mid-IR |
| Uniaxial strain in monolayer TMDs | HSEα + SOC | Valley-resolved band structure on distorted Brillouin zones |
| Valley drift in strained WSe₂ | HSEα + SOC | Differential K/K′ shift producing quasi-indirect gap |
| Rare-earth monopnictides for THz | DFT | ErAs/LuBi nanoparticle–bismide interfaces for THz photoconductors |
